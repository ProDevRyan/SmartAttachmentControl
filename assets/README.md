# Assets Folder

This folder contains preview images and other assets for the Smart Attachment Control documentation.

## Required Assets

- **preview.png**: Screenshot of the control showing the drag-and-drop interface, file list, and styling options

To add a preview image:
1. Take a screenshot of the control in action
2. Save it as `preview.png` in this folder
3. The README.md will automatically reference it

## Recommended Preview Image Content

Your preview image should show:
- The drag-and-drop zone with the upload icon
- A few uploaded files in the list
- The delete icon on file items
- The overall styling and layout

Example dimensions: 800x600 pixels or similar aspect ratio
